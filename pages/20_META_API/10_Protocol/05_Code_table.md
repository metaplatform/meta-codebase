| Code | Description |
| ---- | ----------- |
| **Request** ||
| 101 | Method call |
| 102 | Subscribe |
| 103 | Unsubscribe |
| 104 | Publish |

| 201 | Method result |
| 202 | Subscription confirmation |
| 203 | Unsubscribe confirmation |
| 204 | Publish confirmation - message has been realiable equeued |
| 205 | Incoming published message |
| 206 | Authorization successfull |

| 500 | Generic error |
